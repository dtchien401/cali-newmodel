# Membership Transition Page

## Structure

- `index.html` = page layout
- `assets/css/styles.css` = styling
- `assets/js/data.js` = package/access/benefit/pricing data
- `assets/js/i18n.js` = VI/EN toggle
- `assets/js/app.js` = render and pricing logic

## Run locally

Just open `index.html` in browser.

## Deploy on GitHub Pages

1. Push all files to your repo
2. Go to **Settings**
3. Open **Pages**
4. Source = branch `main` / folder `/root`
5. Save

## Notes

This version uses:
- configurable membership flow
- bilingual UI
- pricing simulation based on current file logic
- light premium background
